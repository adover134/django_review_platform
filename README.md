# 프로젝트 구조
- DBs: Django 백엔드 앱
- customForms: 리뷰 작성 및 회원 정보 입력 용 form
- static: 프론트엔드 구성 파일
- templates: 웹 페이지 html
- webPages: Django 프론트엔드 앱
- webTests: 기능 테스트용 앱 (디버깅 전용)
