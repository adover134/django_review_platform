# 프로젝트 구조
- DBs: Django 백엔드 앱
- customForms: 리뷰 작성 및 회원 정보 입력 용 form
- static: 프론트엔드 구성 파일
- templates: 웹 페이지 html
- webPages: Django 프론트엔드 앱
- webTests: 기능 테스트용 앱 (디버깅 전용)

---
# 주요 기술 스택
- 백엔드
  - Django
  - Django Restframework
  - social_django
- 프론트엔드
  - Kakao Login API
  - Kakao Map API
  - Django
- AI 모델
  - NLP
  - Tensorflow

---
# 주요 기능
- 리뷰 데이터 수집 및 전처리
- 주소 기반 리뷰 지역 표시
- 카테고리 별 키워드 강조
- 소셜 로그인 (Kakaotalk)
